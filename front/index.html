A React2Word facilita a escrita de reações químicas para o Microsoft Word, permitindo o envio de uma ou várias reações químicas com apenas um clique.

Insira uma Reação Química
→

Ou para mais de uma reação, faça upload de um arquivo CSV:

  • Insira os componentes de cada reação química em linhas separadas.
  • Na primeira coluna, insira o coeficiente estequiométrico; na segunda coluna, insira a fórmula molecular.
  • Indique o valor negativo do coeficiente para reagentes e positivo para produtos.
  • Organize as reações em sequência, listando os componentes em linhas consecutivas, conforme abaixo.
  • Após, salve o arquivo no formato CSV.
